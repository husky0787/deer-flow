     - `CardDescription`（pl-8 text-xs）显示 `getFileExtensionDisplayName(file.path) + " file"`
     - `CardAction` 包含下载按钮（保留，用户决定）和 `.skill` 安装按钮（保留，用户决定）

   - 使用 `<a href={urlOfArtifact({ filepath: file.path, threadId, download: true })} target="_blank">`
   - 内含 `<Button variant="ghost"><DownloadIcon className="size-4" /></Button>`
   - `onClick={(e) => e.stopPropagation()}` 防止冒泡触发 selectFile

5. **.skill 安装按钮**：
   - 条件：`file.path.endsWith(".skill")`
   - 使用与 ArtifactFileList 相同的 `installSkill` 逻辑：
     ```typescript
     const [installingFile, setInstallingFile] = useState<string | null>(null);
     const handleInstallSkill = useCallback(async (e: React.MouseEvent, filepath: string) => {
       e.stopPropagation();
       e.preventDefault();
       if (installingFile) return;
       setInstallingFile(filepath);
       try {
         const result = await installSkill({ thread_id: threadId, path: filepath });
         if (result.success) toast.success(result.message);
         else toast.error(result.message || "Failed to install skill");
       } catch (error) {
         console.error("Failed to install skill:", error);
         toast.error("Failed to install skill");
       } finally {
         setInstallingFile(null);
       }
     }, [threadId, installingFile]);
     ```
   - 按钮图标：installingFile 时 `<LoaderIcon className="size-4 animate-spin" />`，否则 `<PackageIcon className="size-4" />`

6. **高亮选中**（FILE-03）：
   - Card `onClick={() => selectFile(file.path)}`（注意：不传 isAutoSelect，默认 false，会停止 autoSelect）
   - Card className 条件样式（Claude's Discretion）：
     ```typescript
     cn(

       selectedFile === file.path
         ? "border-primary bg-primary/5 ring-1 ring-primary/20"
         : "hover:bg-muted/50",
     )
     ```

   - Card ref 回调：`ref={(el) => { if (el) itemRefs.current.set(file.path, el); else itemRefs.current.delete(file.path); }}`
   - `useEffect` 监听 `selectedFile` 变化，调用 `itemRefs.current.get(selectedFile)?.scrollIntoView({ behavior: "smooth", block: "nearest" })`
