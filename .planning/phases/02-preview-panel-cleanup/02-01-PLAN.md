**1. preview-skeleton.tsx (~15 行)**
- `"use client"` 指令
- 使用 `Skeleton` 组件 (`@/components/ui/skeleton`) 模拟文件预览的加载状态
- 布局：顶部一行短条（模拟文件名），中间区域 3-4 行不同宽度的条（模拟内容）

**2. preview-header.tsx (~80 行)**
- `"use client"` 指令
- Props: `{ filepath: string; threadId: string; viewMode: "code" | "preview"; onViewModeChange: (mode: "code" | "preview") => void; isSupportPreview: boolean; content: string | null; }`
- 头部布局参考 `ArtifactFileDetail` 的 header 结构，但不用 Artifact 容器组件，直接用 div：
  - 左侧：`getFileIcon(filepath)` + `getFileName(filepath)` 文件名
  - 中间：当 `isSupportPreview` 为 true 时显示 `ToggleGroup`（code/preview 切换），使用 `Code2Icon` 和 `EyeIcon`
  - 右侧操作栏：
    - `.skill` 文件显示安装按钮（复用 LeftPanel 中的 `installSkill` 逻辑）
    - 复制按钮（`CopyIcon`）：仅 isCodeFile 时显示，调用 `navigator.clipboard.writeText(content)`
    - 下载按钮（`DownloadIcon`）：链接到 `urlOfArtifact({ filepath, threadId, download: true })`
- 使用 `useI18n()` 获取翻译文本，使用 `Tooltip` 组件包裹按钮
- 整体高度固定 h-12，底部 border-b 分隔

**3. file-preview.tsx (~120 行)**
- `"use client"` 指令
- Props: `{ filepath: string; threadId: string; }`
- 使用 `useArtifactContent({ threadId, filepath, enabled: isCodeFile })` 加载文本内容
- 内部状态：`viewMode` ("code" | "preview")，当 `isSupportPreview` 时默认 "preview"，否则 "code"
- `isSupportPreview` 逻辑：`language === "markdown" || language === "html"`（与 ArtifactFileDetail 一致）
- `.skill` 文件视为 markdown（`language = "markdown"`），与 ArtifactFileDetail 一致

文件类型分支渲染逻辑（注意：`checkCodeFile` 对图片/视频返回 `isCodeFile: false`）：

```
if isCodeFile:
  if isSupportPreview && viewMode === "preview":
    if language === "markdown":
      <Streamdown> 渲染（使用 streamdownPlugins）
    if language === "html":
      <iframe src={urlOfArtifact({ filepath, threadId, isMock })}> 预览
  else (viewMode === "code"):
    <CodeBlock code={content} language={language}> Shiki 高亮只读
else (非代码文件):
  根据 getFileExtension(filepath) 判断：
  - 图片扩展名 (jpg/jpeg/png/gif/bmp/webp/svg/ico/tiff/heic):
    <img src={urlOfArtifact({ filepath, threadId, isMock })} className="object-contain max-h-full max-w-full" />
  - 视频扩展名 (mp4/mov/m4v):
    <video src={urlOfArtifact({ filepath, threadId, isMock })} controls muted playsInline className="max-h-full max-w-full" />
  - 其他（纯文本 fallback）:
    使用 fetch 从 urlOfArtifact 加载内容，以 `<pre className="font-mono text-sm whitespace-pre-wrap p-4">` 展示
```

- 图片/视频容器使用 `flex items-center justify-center` 居中
- 使用 `useThread()` 获取 `isMock` 用于 URL 构建
- 使用 `useEffect` 在 `isSupportPreview` 变化时重置 `viewMode`（参考 ArtifactFileDetail 同样逻辑）

**重要：使用 Shiki CodeBlock（`@/components/ai-elements/code-block`）而非 CodeMirror CodeEditor，这是用户锁定决策。**

纯文本 fallback 的内容加载：对于 `isCodeFile === false` 且非图片/非视频的文件，创建一个小的 `usePlainTextContent` hook（可以内联在组件中），使用 `useQuery` 从 `urlOfArtifact` fetch 文本内容，staleTime 5分钟，与 `useArtifactContent` 保持一致。

图片/视频扩展名判断：创建辅助函数 `isImageFile(ext)` 和 `isVideoFile(ext)`，扩展名列表参考 `getFileIcon` 中的 switch case。
